# Project Summary
The project is a dynamic visualization dashboard that analyzes Brent crude oil price predictions alongside various economic indicators. By leveraging advanced AI technologies, it provides real-time insights through interactive data analysis and chat functionalities. This tool aggregates diverse data sources, enabling stakeholders in the oil industry to gain a comprehensive overview of historical trends, facilitating informed decision-making in a volatile oil market.

# Project Module Description
- **Frontend**: A responsive web interface built with React, featuring components such as charts, news panels, and AI chat functionalities.
- **Backend**: Manages data processing, storage, and interactions with AI models to ensure real-time updates and analysis.

# File Description Inventory
- **app/backend/**: Contains backend logic, including models, routers, and services for data processing and API requests.
- **app/frontend/**: Contains frontend logic, including components, pages, and styles for the user interface.
- **uploads/**: Stores images and other assets used in the application.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Recharts
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **AI**: Custom AI model integration using deepseek-v3.2

# Usage
1. **Install Dependencies**:
   - For the backend: Run `pip install -r requirements.txt` within the `app/backend` directory.
   - For the frontend: Run `pnpm install` within the `app/frontend` directory.

2. **Build the Project**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run the Application**:
   - Start the backend server.
   - Start the frontend application.
